## \<Patcher\>

Open in your browser and deploy with pleasure

### Version
Alpha: `v0.9`

### Download

```
$ git clone git@github.com:likerRr/patcher.git
```

### How to
* choose project
* choose date of project last update time
* create patch
* check patch (depends on git patches)
* update server (manually)
